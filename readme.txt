 
============================
AppLovin iPhone SDK Demo App
============================

This project provides examples of AppLovin SDK integration.
For more information, please visit http://applovin.com/integration.

-------------------------------
Using an ad mediation platform?
-------------------------------

Please note that while this demo project includes example integrations with other networks via our adaptor code, this repository does not include the adaptors themselves. Those files are located at https://github.com/AppLovin/SDK-Network-Adaptors/.
